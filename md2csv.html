How to Use Table Multi-format Editor & Converter

What This Tool Does

Multi-format Editor & Converter creates and converts tables between different formats: Markdown, CSV, JSON, and HTML. Tables can be input, edited, or exported in any format, and all formats update in real time.

Getting Started

Click any section to edit directly in that format. All other formats will update automatically, and you can switch between them at any time.

Quick Examples

CSV: Name,Age,City
John,25,NYC
Jane,30,LA

Markdown: | Name | Age | City |
|------|-----|------|
| John | 25 | NYC |

JSON: [["Name","Age","City"],["John","25","NYC"]]

How to Use Each Section

  • Preview: Preview the rendered table
  • Markdown: Format used in ChatGPT and Github
  • CSV: Comma-separated values for spreadsheet import/export
  • JSON: Format used in Restful APIs. (JSON objects not used because they are not 100% compatable with all data)
  • HTML: Code for tables used in web pages

Basic Actions

  • Edit: Click any section to edit in that format
  • Clear: Use the Clear button to delete the table and start over
  • Copy Icon: Copies the table in that format to be pasted elsewhere
  • Expand/Collapse: Click section headers to show/hide sections
